# Akbar Makhmanazarov - Cyber Security Portfolio

A world-class, professionally structured portfolio website showcasing expertise in cybersecurity, penetration testing, network security, and IT infrastructure.

## Features

- **Modern Architecture**: Modular CSS and JavaScript for easy maintenance
- **Cybersecurity Theme**: Hacker-style green Matrix design with terminal aesthetics
- **Performance Optimized**: Smooth animations with hardware acceleration
- **Fully Responsive**: Adaptive design for all devices and screen sizes
- **SEO Optimized**: Complete meta tags, Open Graph, and structured data
- **Accessibility**: ARIA labels and semantic HTML

## Tech Stack

### Frontend
- HTML5 with semantic markup
- Modular CSS (Variables, Components, Layout, Animations, Responsive)
- Vanilla JavaScript (ES6+ modules)
- Font Awesome icons
- AOS (Animate On Scroll) library

### Key Technologies
- **Cyber Security**: Kali Linux, Wireshark, NMAP, BurpSuite, Wifite, Metasploit
- **Networking**: Cisco, TCP/IP, Network Security, Firewall, VPN
- **Programming**: Python, C#, C++, JavaScript
- **Web Development**: HTML5, CSS3, React, Node.js

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/Akbar2998/Akbar2998.github.io.git
cd Akbar2998.github.io
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser
